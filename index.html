Study

The specialty “System programming” is wide profile and contains all technical and technological problems, connected with operation and maintenance of the big variety from computer and microprocessor technology and the constantly increasing number of computer networks and Internet. The main goal of the training is preparing the students for their realization of the acquired specialty in the area of information and communications technology.

Object Orientated
Programming

Object oriented programming – job with classes and objects, of fundamental principles of object oriented programming (OOP) and how to use them in the practice. The training aims to mastery of paradigms of OOP and their use for modeling of tasks from the real world.

Assembler Languages

Programming on assembly language-Assembler use addresses instead of recording memory cells specific label. They process the code, transfer him so-called directive. This relative addresses that do not affect CPU (not translated on machine language), as required for the recognition of programming.

Web Design

Web design is arranging the elements of one site – hypertext (links). Computer graphics, computer animation, sound, specific way for presenting the site after that from a web browser, most-commonly from a visual way. The web design includes creating one or more pages, which are organized through set of inner references (hyperlinks).

Languages

Pascal

Pascal is an imperative and procedural programming language, created by Niklaus Wirth in 1970 from the Swiss national technology institute. It’s basic purpose was to be used for teaching, i.e to be easy to learn and to teach on good style of programming. The language was named after the French mathematician Blaise Pascal.

Visual Basic

Visual Basic is an event-driven programming language, based on the language BASIC. It is developed by Microsoft Corporation and is a part of the package for development Visual Studio. This is the main environment of Microsoft’s technique RAD. With her it’s easy to develop applications with graphical interface (GUI) for OC Windows. It provides a quick acces to data base, using DAO,ADO,RDO. It is based on ActiveX software components.

C++

C++ is non-specialized language from high level. He is object-oriented language with static types. From 1990, C++ is one of the most popular commercial languages for programming. Bjarne Stroustrup develops C++ in 1983 in Bell laboratories as an extension language C - language is based on C, but it added a number of additional options and made some changes.

C#

C# is object-oriented language for programming, developed by Microsoft, like part of the software platform .NET. The aspiration from the begining of the creation of C# was to create one easy, modern, object-oriented language with general use. The first version of C# was developed by Microsoft in the period 1999-2002 and is official released in use in 2002, like part from .NET platform.

HTML

HTML is the basic language for design for web pages. HTML is a standart in Internet. The current version of the standart is HTML 5.0 (since 28 october 2014.). The description of the document is made by special elements, named HTML elements or markers, which are based of labels and tags (HTML tags) and angle brackets.

CSS

CSS (Cascading Style Sheets) is a language for description of a document, written in a markup language. The most commonly used together with HTML, but may be applied to any XML document. Official specifications for CSS supported from World Wide Web Consortium. Using CSS, options for formatting can be placed in one file, and then the change will be reflected on all pages, which use this CSS file.
